# 情报分析技能

## 分析框架

### 1. 核心发现
**关键数据点：**
- 最新数据：入住率、价格、流量
- 重要事件：政策变化、竞品动作
- 突发情况：行业黑天鹅事件

**核心观点：**
- 数据对比：同比、环比
- 趋势判断：上升/下降/稳定
- 重要性排序：哪些最值得关注

### 2. 行业趋势
**政策变化：**
- 监管动态：新规定、新标准
- 政府支持：优惠政策、补贴
- 行业规范：自律公约、标准制定

**消费趋势：**
- 需求变化：消费者偏好
- 消费升级：品质要求提升
- 新兴市场：细分客群崛起

**技术发展：**
- 智能化：AI、大数据应用
- 数字化：线上线下融合
- 创新模式：共享、订阅等

### 3. 竞品分析
**优势分析：**
- 价格策略：是否有价格优势
- 服务特色：哪些服务做得好
- 营销方式：推广渠道、内容风格
- 用户评价：好评点、差评点

**劣势分析：**
- 痛点问题：运营缺陷
- 用户不满：差评分析
- 市场反应：负面事件

**策略借鉴：**
- 可以学习的优点
- 需要避免的教训

### 4. 行动建议
**短期调整（1-3个月）：**
- 紧急应对：针对突发情况
- 快速优化：发现的问题
- 临时策略：应对市场变化

**长期规划（3-12个月）：**
- 战略调整：根据趋势变化
- 能力建设：补齐短板
- 新业务布局：抓住新机会

**风险提示：**
- 潜在风险：可能面临的问题
- 应对预案：如何防范
- 监控指标：需要关注的数据

## 联网搜索策略

### 1. 搜索关键词组合
**行业搜索：**
- "民宿行业 2026 趋势"
- "长沙民宿 市场分析"
- "民宿 政策 最新"

**竞品搜索：**
- "长沙 头部民宿 品牌"
- "途家 民宿 价格 策略"
- "美团民宿 热门房源 长沙"

**用户搜索：**
- "民宿 用户需求 变化"
- "旅行 消费趋势 2026"
- "年轻人 住宿 偏好"

### 2. 数据来源优先级
**权威媒体：**
- 行业媒体：执惠、迈点
- 政府网站：文旅部门
- 研究机构：艾瑞、易观

**平台数据：**
- OTA 平台：途家、美团、飞猪
- 社交媒体：小红书、抖音、微博
- 评论网站：大众点评、携程

**行业报告：**
- 年度报告：行业白皮书
- 月度报告：数据监测
- 专项报告：深度分析

### 3. 信息验证
**多源交叉：**
- 同一事件，至少 2 个来源
- 数据对比，排除异常值
- 专家观点，交叉验证

**时效性检查：**
- 优先选择最新信息
- 注明数据时间范围
- 过期数据慎用

## 输出格式规范

### 1. 禁用表格
- ❌ 不要使用任何表格
- ❌ 不要使用 Markdown 表格语法
- ❌ 不要使用三线表
- 所有数据用纯文本或列表呈现

### 2. 数据呈现方式
**对比数据：**
```
途家平台：平均价格 350元，入住率 75%
美团平台：平均价格 320元，入住率 80%
飞猪平台：平均价格 380元，入住率 70%
```

**趋势数据：**
```
入住率变化：
- 3月：72%
- 4月：78%（环比上升 6%）
- 5月预计：85%（五一假期拉动）
```

**列表数据：**
```
主要竞品：
1. A民宿：价格优势明显，服务评价高
2. B民宿：地理位置优越，装修豪华
3. C民宿：特色主题房间，年轻客群多
```

### 3. 引用标注
- 信息来源：注明出处
- 数据时间：标注时间范围
- 可靠性：说明可信度

## 分析质量标准

### 1. 准确性
- 数据来源可靠
- 分析逻辑严密
- 结论有据可依

### 2. 时效性
- 信息及时更新
- 趋势判断准确
- 预测合理

### 3. 实用性
- 建议可执行
- 优先级明确
- 风险可控

### 4. 完整性
- 覆盖主要维度
- 不遗漏关键信息
- 提供多角度分析
