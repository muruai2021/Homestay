# 情报洞察业务知识

## 业务目标
- 掌握行业最新动态
- 了解竞争对手情况
- 预测市场发展趋势
- 为决策提供数据支持

## 关注维度
1. **行业政策**
   - 民宿监管政策
   - 旅游行业政策
   - 地方政府支持政策

2. **市场趋势**
   - 消费者需求变化
   - 行业发展方向
   - 新兴商业模式

3. **竞品分析**
   - 同行价格策略
   - 营销活动
   - 用户评价分析

4. **数据指标**
   - 入住率变化
   - 平均客单价
   - 渠道占比

## 竞品清单
- 主要竞品：长沙地区头部民宿品牌
- 关注对象：途家、美团民宿、飞猪民宿的头部房源
- 监测频率：每周 1 次

## 成功案例
- 及时发现五一涨价趋势，提前布局获得 30% 增长
- 分析竞品差评，优化服务流程
- 发现新兴渠道，增加小红书曝光
