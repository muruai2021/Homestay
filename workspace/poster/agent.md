# 海报生成技能配置

## 技能概述
使用AI图片生成模型为民宿业务创建高质量宣传海报。

## 工作流程

### 1. 理解需求
- 接收用户的主题描述
- 分析海报的使用场景（节日促销、房源推荐、活动宣传等）
- 确定目标受众和风格偏好

### 2. 构建提示词
根据不同的海报类型，构建专业的提示词：

#### 节日促销类
```
主题：五一假期特惠、国庆黄金周、春节返乡等
风格：现代简约 / 复古怀旧 / 活力运动
元素：优惠信息、时间、联系方式、二维码占位
色调：红色系（喜庆）或金色系（高端）
```

#### 房源推荐类
```
主题：精选房源推荐、宝藏民宿发现、网红打卡地
风格：温馨舒适 / 高端奢华 / 科技感
元素：房型信息、价格区间、特色标签
色调：暖色调（温馨）或冷色调（高端）
```

#### 活动宣传类
```
主题：开业庆典、周年庆、新品发布、会员活动
风格：现代简约 / 商务专业 / 创意艺术
元素：活动信息、时间地点、参与方式
色调：品牌色系（保持一致）
```

#### 主题海报类
```
主题：城市探索、长沙地标、橘子洲头等、岳麓书院等
风格：文艺清新 / 复古怀旧 / 现代极简
元素：景点名称、特色描述、打卡引导
色调：自然色调（清新）
```

### 3. 生成参数设置

#### 尺寸规格
- **竖版海报**：1024x1792（3:4比例）
  - 用途：手机分享、朋友圈、小红书
  - 适合：房源展示、个人推广
  
- **横版海报**：1792x1024（16:9比例）
  - 用途：横幅、网站Banner、PPT背景
  - 适合：活动宣传、品牌展示

- **方形海报**：1024x1024（1:1比例）
  - 用途：缩略图、头像、社交媒体
  - 适合：品牌Logo、APP图标

#### 质量设置
- **标准质量**：生成速度快，适合预览和快速使用
- **高清质量**：细节丰富，适合印刷和高质量展示
- **超清质量**：细节极致，适合大型活动和专业印刷

### 4. 风格定义

#### 现代简约
```
特点：简洁线条、留白充足、色彩克制
适用场景：高端民宿、商务活动
色调搭配：黑灰白 + 一个主色
```

#### 复古怀旧
```
特点：暖色调、胶片质感、字体传统
适用场景：老字号民宿、历史文化景点
色调搭配：米黄、砖红、古铜绿
```

#### 科技感
```
特点：冷色调、几何图形、渐变效果
适用场景：智能民宿、科技主题房间
色调搭配：深蓝、紫、银灰
```

#### 温馨舒适
```
特点：暖色调、柔和线条、圆润图形
适用场景：家庭民宿、亲子主题
色调搭配：米白、浅粉、鹅黄
```

#### 高端奢华
```
特点：金色系、光泽效果、精致细节
适用场景：精品民宿、VIP活动
色调搭配：金、黑、深棕
```

#### 活力运动
```
特点：鲜艳色彩、动感线条、运动元素
适用场景：运动主题民宿、户外活动
色调搭配：橙、红、明黄
```

## 生成输出格式

### 输出要求
1. **图片格式**：PNG（保持高质量）
2. **图片比例**：根据场景选择（默认3:4）
3. **分辨率**：最低1024px宽
4. **命名规范**：{类型}_{主题}_{时间戳}.png

### 文件保存
- **保存路径**：workspace/海报生成智能体/outputs/
- **命名示例**：promotion_五一特惠_20240407_143022.png
- **元数据**：记录提示词、风格、生成时间

## 质量标准

### 海报质量检查清单
- ✅ 图像清晰，文字可读
- ✅ 色彩协调，风格统一
- ✅ 重点突出，层次分明
- ✅ 信息完整，要素齐全
- ✅ 符合场景，目标明确
- ✅ 专业美观，具有传播性

### 不合格标准
- ❌ 文字模糊，难以识别
- ❌ 色彩冲突，风格混乱
- ❌ 信息过多，没有重点
- ❌ 质量过低，不够专业
- ❌ 偏离需求，不符合场景

## 优化建议

### 提高生成效果
1. **具体描述**：提供详细的主题和元素要求
2. **风格明确**：明确指定想要的艺术风格
3. **用途说明**：说明海报的使用场景
4. **参考风格**：提供喜欢的参考或风格描述
5. **分步生成**：复杂海报可以分步生成多个元素

### 常见问题处理
1. **文字模糊**：增加字体大小、提高分辨率
2. **颜色不准确**：更具体地描述颜色
3. **元素缺失**：明确列出需要包含的所有元素
4. **风格不符**：提供风格参考或关键词
5. **生成失败**：重试或简化描述

## 技术说明

### AI模型配置
- **模型**：DALL-E 3 / Stable Diffusion / FLUX
- **分辨率**：1024x1024 / 1792x1024 / 1024x1792
- **格式**：PNG（无损）
- **质量**：standard / hd

### 性能优化
- 生成时间：通常5-15秒
- 重试机制：失败自动重试1-2次
- 超时处理：超过30秒自动取消
