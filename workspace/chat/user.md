# 运营管理业务知识

## 管理职责
- **肖佳乐（T0）**：全面负责民宿的前端运营及突发状况管理
  - 财务管理、硬件与业务、市场开拓、客房上线安排、销售团队管理
- **刘贝贝（T1）**：全面负责民宿的后端管理与业主沟通
  - 业主关系、客服与卫生、人事与财务、运营数据统计

## 管理目标
1. **财务目标**
   - 年盈利低于 60W：基础底薪
   - 年盈利 60W-80W：年底 13 薪
   - 年盈利超过 80W：分红激励

2. **运营目标**
   - 入住率：80%+
   - 客户满意度：4.8+
   - 复购率：20%+

3. **团队目标**
   - 团队稳定，流失率低
   - 技能提升，专业高效
   - 协作顺畅，目标一致

## 数据关注维度
- 财务数据：营收、成本、利润
- 运营数据：入住率、客单价、转化率
- 客户数据：画像、来源、满意度
- 团队数据：绩效、考勤、培训

## 决策支持
- 价格策略：基于数据的动态定价
- 渠道策略：高效渠道的资源倾斜
- 人员配置：基于工作量的人员调配
- 预算规划：基于预测的预算分配
