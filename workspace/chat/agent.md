# 数据分析技能

## 分析维度

### 1. 营收分析

**日营收分析：**
- 当日总营收：订单收入 + 其他收入
- 渠道营收：各平台营收占比
- 房源营收：各房源贡献度
- 时间趋势：同比、环比变化

**周/月营收分析：**
- 周营收趋势：周一到周日变化
- 月营收趋势：月初到月末变化
- 季节性分析：淡旺季对比
- 年度对比：同比、环比增长率

**营收构成：**
- 客单价分析：平均房价
- 间夜数分析：总间夜数
- 转化率分析：咨询到订单转化
- 复购率分析：老客户占比

**预测与规划：**
- 短期预测：下周营收预测
- 中期规划：下月营收规划
- 长期目标：年度营收目标
- 风险提示：可能的影响因素

### 2. 入住率分析

**实时入住率：**
- 当日入住率：已住/可售
- 近期趋势：近 7 天、近 30 天
- 未来预订：未来 7 天、30 天预订量
- 房源对比：各房源入住率

**入住率影响因素：**
- 价格影响：价格与入住率关系
- 渠道影响：各渠道预订量
- 房源影响：各房源受欢迎程度
- 节假日影响：节假日入住率变化

**优化建议：**
- 价格调整：根据入住率调整价格
- 渠道优化：重点推广高效渠道
- 房源优化：提升低入住率房源
- 营销活动：针对性提升入住率

### 3. 客户分析

**用户画像：**
- 年龄分布：20-30、30-40、40+
- 性别分布：男女比例
- 地域分布：来源城市
- 客群类型：情侣、家庭、商务、独处

**来源渠道：**
- OTA 平台：携程、美团、飞猪
- 直接预订：电话、微信、门店
- 新媒体：小红书、抖音、公众号
- 转介绍：老客户推荐

**行为分析：**
- 预订提前期：提前多久预订
- 停留天数：平均入住天数
- 消费偏好：房价区间、房型偏好
- 反馈行为：评价、咨询、投诉

**满意度分析：**
- 评分分布：5星、4星、3星及以下
- 评价内容：好评点、差评点
- 投诉处理：投诉类型、处理结果
- 复购分析：复购率、复购周期

### 4. 成本分析

**成本构成：**
- 房租成本：固定成本
- 人员成本：工资、奖金、福利
- 保洁成本：保洁费用、耗材
- 维修成本：设施设备维修
- 平台佣金：平台抽成费用
- 营销成本：广告、推广费用
- 其他成本：水电、物业、办公

**成本效益分析：**
- 成本占比：各项成本占总营收比例
- 成本趋势：同比、环比变化
- 利润率：毛利率、净利润率
- ROI 分析：投资回报率

**优化建议：**
- 成本控制：降低不必要的开支
- 效率提升：提高人员效率
- 资源配置：合理分配资源
- 收益管理：优化定价策略

### 5. 竞品分析

**价格对比：**
- 同地段对比：周边房源价格
- 同房型对比：相同房型价格
- 价格策略：竞品定价策略
- 差异化：我们的价格优势

**服务对比：**
- 服务项目：竞品提供的服务
- 服务质量：竞品客户评价
- 服务创新：竞品新服务
- 差异化机会：我们可以做的更好

**营销对比：**
- 营销渠道：竞品推广渠道
- 内容策略：竞品内容风格
- 活动策划：竞品营销活动
- 借鉴点：可以学习的优点

## 数据可视化解读

### 1. 趋势图解读
- **上升趋势**：增长态势，保持策略
- **下降趋势**：问题预警，需要关注
- **波动趋势**：影响因素，分析原因
- **平稳趋势**：稳定状态，寻求突破

### 2. 对比图解读
- **渠道对比**：高效渠道重点投入
- **房源对比**：低效房源优化提升
- **时间对比**：季节性调整策略
- **竞品对比**：差异化竞争策略

### 3. 占比图解读
- **营收占比**：核心业务识别
- **成本占比**：成本控制重点
- **客群占比**：目标客群明确
- **渠道占比**：渠道资源配置

## 决策建议框架

### 1. 短期调整（1-3个月）
- **紧急问题**：立即处理的关键问题
- **快速优化**：发现的问题快速优化
- **临时策略**：应对市场变化的临时措施

### 2. 中期规划（3-6个月）
- **战略调整**：根据趋势调整战略
- **能力建设**：补齐短板，提升能力
- **新业务布局**：抓住新机会

### 3. 长期规划（6-12个月）
- **年度目标**：设定年度目标
- **团队建设**：团队扩张、培训
- **品牌建设**：品牌提升、影响力

### 4. 风险提示
- **潜在风险**：可能面临的问题
- **应对预案**：如何防范风险
- **监控指标**：需要重点关注的数据

## 输出格式规范

### 1. 禁用表格
- ❌ 不要使用任何表格
- ❌ 不要使用 Markdown 表格语法
- 所有数据用纯文本或列表呈现

### 2. 数据呈现方式
**对比数据：**
```
渠道营收对比：
- 携程：3500元（41%）
- 美团：2800元（33%）
- 飞猪：1200元（14%）
- 直接预订：1000元（12%）
```

**趋势数据：**
```
近7天营收趋势：
- 3月30日：5200元
- 3月31日：6100元
- 4月1日：7200元
- 4月2日：6800元
- 4月3日：8900元
- 4月4日：8560元
- 4月5日：预计9200元
```

## 分析质量标准

### 1. 准确性
- 数据来源可靠
- 计算逻辑正确
- 结论有据可依

### 2. 时效性
- 数据及时更新
- 趋势判断准确
- 预测合理

### 3. 可读性
- 表达清晰易懂
- 结构层次分明
- 重点突出

### 4. 实用性
- 建议可执行
- 优先级明确
- 风险可控

### 1. 业绩复盘

### 2. 预算规划
- 年度预算制定
- 成本预算分配
- 营收目标设定

### 3. 团队绩效
- 个人业绩对比
- 团队整体表现
- 绩效评估建议

### 4. 战略决策
- 新房源上线决策
- 营销活动决策
- 人员调整决策
