# 文件上传与 Markdown 转换技能

## 技能概述

本技能用于处理用户上传的各类文档，自动转换为 Markdown 格式并保存到知识库。

## 支持的文件格式

| 格式 | 扩展名 | 处理方式 |
|------|--------|----------|
| PDF | .pdf | PyMuPDF/PDFMiner 提取文本 |
| Word | .docx, .doc | python-docx 解析 |
| 文本 | .txt | 直接读取 |
| Markdown | .md | 直接读取 |

## 转换流程

### 1. 文件接收
- 接收用户上传的文件
- 验证文件格式和大小
- 生成唯一文件名

### 2. 内容提取
- 根据文件类型调用对应的解析器
- 提取文本内容和结构信息
- 处理编码问题（中文优先 UTF-8）

### 3. Markdown 转换
- 标题层级转换
- 列表结构转换
- 表格转换
- 链接和图片处理

### 4. 保存存档
- 保存到知识库目录
- 更新元数据索引
- 返回保存结果

## 目录结构

```
workspace/民宿服务小助手/
├── knowledge/
│   ├── 政策文件/
│   ├── 运营手册/
│   ├── 培训资料/
│   └── 其他资料/
├── metadata.json
└── uploads/  # 原始文件备份
```

## 错误处理

- 文件格式不支持：提示用户并列出支持格式
- 文件过大：提示大小限制
- 解析失败：返回原始错误信息
- 存储失败：提示存储空间问题
