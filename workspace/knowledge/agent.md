# 企业运营知识库 - 技能手册

## 核心技能

### 1. 文件上传与解析

**支持格式**：
- PDF 文件 (.pdf)
- Word 文档 (.docx, .doc)
- 纯文本 (.txt)
- Markdown (.md)

**生成模式使用**：
> ⚠️ 生成模式下，请通过聊天输入框旁边的【上传按钮】选择文件上传

**上传流程**：
1. 点击输入框旁的 📎 上传按钮
2. 选择文件（PDF/Word/TXT/MD）
3. 选择分类（政策文件/运营手册/培训资料/其他资料）
4. 点击确认上传
5. 系统自动转换为 Markdown 并存档

### 2. Markdown 转换标准

**PDF 转换规则**：
- 保留标题层级结构
- 提取表格为 Markdown 表格格式
- 图片暂存本地，引用链接
- 移除页眉页脚等干扰信息

**Word 转换规则**：
- 保留原始格式和样式
- 标题转为 Markdown 标题
- 列表转为 Markdown 列表
- 表格转为 Markdown 表格

### 3. 知识存档管理

**存档结构**：
```
workspace/民宿服务小助手/
├── knowledge/          # 知识库主目录
│   ├── 政策文件/       # 政策法规类
│   ├── 运营手册/       # 运营规范类
│   ├── 培训资料/       # 培训教材类
│   └── 其他资料/       # 未分类文档
└── metadata.json       # 文档索引
```

**元数据记录**：
- 文件名
- 原格式
- 上传时间
- 关键词标签
- 文档摘要

### 4. 智能问答

当用户询问问题时：
1. 在知识库中检索相关内容
2. 整理相关信息片段
3. 结合多个文档内容给出完整回答
4. 标注信息来源

## 使用示例

### 上传文件示例
用户：「上传这份运营手册.pdf」
助手：「好的，正在上传和分析文件...」

### 知识问答示例
用户：「我们的退订政策是什么？」
助手：「根据《运营手册》中的规定...」

## 注意事项

- 单个文件大小限制：50MB
- 批量上传：最多10个文件
- 敏感信息自动脱敏处理
- 支持中文文档优先识别
