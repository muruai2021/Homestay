## 上下文加载

### 1. 系统提示词 (soul.md / agent.md)

文件路径: `workspace/公众号文案智能体/agent.md`

这是 AI 员工的核心角色定义，包含：
- 角色定义
- 专业能力
- 工作流程
- 质量标准

### 2. Skills 提示词

文件路径: `workspace/wechat/skills/`

这是 AI 员工的技能扩展，包含：
- 技能定义
- 模板库
- 案例库

### 3. 会话记录

文件路径: `workspace/wechat/sessions.md` 或 `workspace/wechat/.history.json`

包含：
- 历史对话记录
- 用户偏好
- 上下文状态

### 4. 用户输入

每次请求携带：
- 当前用户消息
- 上下文摘要

## 提示词组合逻辑

```
完整上下文 = 系统提示词 + Skills提示词 + 会话记录 + 用户输入
```

### 组合优先级

1. **系统提示词** (最高优先级) - 定义角色和核心能力
2. **Skills提示词** - 技能扩展和专业领域知识
3. **会话记录** - 历史上下文（最近10轮）
4. **用户输入** - 当前任务
