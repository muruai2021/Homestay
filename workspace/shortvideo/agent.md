# 短视频脚本创作技能

## 脚本结构标准（30-60秒）

### 1. 开场设计（0-3秒）- 黄金3秒

**视觉冲击：**
- 快速剪辑：3-5个镜头快速切换
- 镜头运动：推拉摇移，动感十足
- 特效：字幕动效、转场特效

**疑问/反常识：**
- "长沙还有这种地方？"
- "3天花1万住民宿？"
- "这家民宿不按套路出牌"

**直接利益点：**
- "离橘子洲步行5分钟！"
- "人均150住江景房！"
- "这服务也太贴心了吧"

**悬念设置：**
- "最后这个细节太绝了..."
- "住完不想走的民宿长这样"
- "90%的人都不知道"

### 2. 主体内容（3-50秒）

**分镜规划（3-5个镜头）：**

**镜头1：全景展示（10-15秒）**
- 画面：房间全景/外景
- 运镜：缓慢推拉/环绕
- 文案："这是我在长沙住过的最特别的民宿..."

**镜头2：细节特写（10-15秒）**
- 画面：软装细节/设施设备
- 运镜：特写镜头
- 文案："每一个细节都精心设计..."

**镜头3：体验展示（10-15秒）**
- 画面：入住体验/服务场景
- 运镜：跟随镜头/主观视角
- 文案："入住流程超方便，还有..."

**镜头4：对比/证明（10-15秒）**
- 画面：数据/客户反馈
- 运镜：快速剪辑
- 文案："95%的客人说会再来..."

**镜头5：亮点总结（5-10秒）**
- 画面：亮点合集
- 运镜：蒙太奇
- 文案："总结一下，这家民宿..."

### 3. 结尾设计（50-60秒）

**金句升华：**
- "住的不是酒店，是生活"
- "猩伙伴，懂你的每一个需求"
- "在长沙，找到属于你的家"

**互动引导：**
- "想看更多？关注我！"
- "需要攻略？私信我！"
- "评论区聊聊你的看法"

**行动引导：**
- "点击下方链接预订"
- "私信获取优惠"
- "收藏备用，下次来长沙就用它"

### 4. 背景音乐建议

**轻快温馨风：**
- 推荐曲目：轻音乐、民谣
- 节奏：中等，不抢戏
- 情绪：温暖、治愈

**动感活力风：**
- 推荐曲目：流行音乐、电子乐
- 节奏：较快，有感染力
- 情绪：兴奋、期待

**高级质感风：**
- 推荐曲目：钢琴曲、古典乐
- 节奏：缓慢，优雅
- 情绪：优雅、品味

## 不同类型脚本模板

### 1. 房间展示型
```
0-3秒：全景快剪，配音乐
3-20秒：从外到内，逐步展示
20-40秒：细节特写，突出亮点
40-55秒：服务流程，增强信任
55-60秒：金句总结+引导关注
```

### 2. 服务体验型
```
0-3秒：问题抛出，引发好奇
3-20秒：服务场景，真实记录
20-40秒：客户反馈，增强信任
40-55秒：对比效果，突出优势
55-60秒：互动引导，私信咨询
```

### 3. 干货分享型
```
0-3秒：痛点引入，产生共鸣
3-20秒：方法讲解，逻辑清晰
20-40秒：实操演示，简单易懂
40-55秒：效果展示，增强说服
55-60秒：总结提醒+收藏引导
```

### 4. 情感故事型
```
0-3秒：冲突开场，抓住注意力
3-20秒：故事展开，情感铺垫
20-40秒：转折/升华，情感释放
40-55秒：价值传递，品牌露出
55-60秒：金句结尾，引发共鸣
```

```
字幕：精致软装

...
```

## 金句库

**情感共鸣类：**
- "住的不是酒店，是生活"
- "在陌生的城市，找到归属感"
- "旅行不只是看风景，更是体验生活"

**价值主张类：**
- "用心服务，让每一次入住都成为美好回忆"
- "猩伙伴，不止是一间民宿"
- "懂你的每一个需求"

**行动号召类：**
- "下次来长沙，一定要体验一下"
- "现在预订，享受五一早鸟价"
- "私信我，获取专属优惠"

## 禁忌事项
❌ 不要开头慢热
❌ 不要信息过载
❌ 不要时长超过60秒
❌ 不要背景音乐抢戏
❌ 不要文案过长

## 质量检查清单
- [ ] 前3秒抓住注意力
- [ ] 3-5个分镜，节奏明快
- [ ] 每个镜头有明确目标
- [ ] 背景音乐匹配情绪
- [ ] 金句易于传播
- [ ] 有清晰的互动引导
- [ ] 时长控制在 30-60 秒
