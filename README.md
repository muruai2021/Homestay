## 🎯 功能介绍

### 1. 首页门户
- 极简风格入口导航，快速访问所有功能模块
- 运营数据预览卡片，实时显示关键指标
- 优雅的卡片动效和渐变设计

### 2. 小红书文案智能体
- **输入**：产品名称、使用场景
- **输出**：150-200字种草文案
- **特点**：带 Emoji 表情、话题标签、手机端友好

### 3. 公众号文章智能体
- **输入**：文章关键词、分类（产品/知识/感悟/故事/品宣）、目标客户群画像
- **输出**：1500-2000字专业公众号文章
- **特点**：标题优化、正文结构化、金句埋点、互动引导

### 4. 短视频文案智能体
- **输入**：视频主题、风格偏好
- **输出**：30-60秒口播脚本
- **特点**：分镜建议、背景音乐推荐、关键台词/金句、3秒开场设计

### 5. 民宿情报洞察智能体（联网搜索）
- **输入**：行业关键词（如"民宿营销策略"）
- **输出**：深度洞察报告
- **特点**：
  - ✨ 基于通义千问联网搜索获取最新信息
  - 📊 行业趋势分析
  - 💡 关键洞察提炼
  - 📋 行动建议输出

### 6. 民宿海报生成智能体
- **输入**：文案内容、图片尺寸（3:4/16:9/1:1/4:3）
- **输出**：AI 生成的海报图片描述词
- **特点**：支持一键生成民宿宣传海报

### 7. 猩伙伴 AI 运营助手
- **功能**：7×24小时智能客服
- **特点**：基于知识库回答、Markdown 格式化输出

### 8. 运营管理驾驶舱
- 今日/本月订单统计
- 营收数据看板
- 入住率分析
- 客户评分追踪
- 订单来源分布（携程/美团/飞猪/直订）

## 🛠️ 技术架构

### 后端
- **框架**：Python + FastAPI
- **AI 引擎**：通义千问（qwen-plus）
- **联网搜索**：enable_search 参数（必应搜索集成）
- **图片生成**：通义万相 API（wan2.6-t2i）
- **部署端口**：9002

### 前端
- **技术**：原生 HTML/CSS/JavaScript
- **Markdown 渲染**：marked.js
- **样式**：自定义 CSS（极简现代风格）
- **响应式**：支持桌面和移动端

### 部署方式
- **静态文件**：内置于后端（StaticFiles），无需单独部署
- **服务器要求**：Windows/Linux，Python 3.10+

## 🚀 快速部署

### 环境要求
- Python 3.10+
- 通义千问 API Key（阿里云百炼平台）

### 步骤一：安装依赖
```bash
pip install -r requirements.txt
```

### 步骤二：配置 API Key
编辑 `.env` 文件：
```
DASHSCOPE_API_KEY=你的API密钥
AI_PROVIDER=qwen
```

### 步骤三：启动服务
```bash
python main.py
```

服务启动后访问：`http://服务器IP:9002`

## ⚙️ 宝塔面板部署

1. **上传项目**到服务器（如 `D:\www\ai-matrix`）
2. **安装 Python 项目管理器**，添加项目：
   - 启动文件：`main.py`
   - 端口：`9002`
3. **安装依赖**：`pip install -r requirements.txt`
4. **添加站点** → **反向代理**：
   - 目标URL：`http://127.0.0.1:9002`
5. **配置域名**（可选）
